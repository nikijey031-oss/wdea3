# Tinyyshaww

Tinyyshaww is a simple e-commerce website developed as a first-semester web development project using HTML, CSS, and JavaScript.

## Features
- Responsive user interface
- Product catalog
- Wishlist functionality
- Login and registration forms
- Modern and clean design

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Purpose
This project was created to learn the fundamentals of front-end web development, including page structure, styling, responsiveness, and DOM manipulation.
